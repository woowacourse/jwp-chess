# 웹 체스
> (e): Exception 처리

### 1단계 요구 사항
- 스프링 애플리케이션으로 체스 실행
- Spring MVC를 활용 (Spark 대체)
- JdbcTemplate 활용 (기존의 Connection 직접 생성 로직 대체)
- 요청 받기: @Controller / @RestController 활용

### 2단계 요구 사항
- 체스방 생성
  - 제목, 비밀번호 입력받기
  - 생성 시 만들어지는 고유값은 url에 활용
- 체스방 목록 조회
  - 제목 클릭 시 체스방 참가
  - 삭제 버튼 클릭 시 체스방 삭제
    - 비밀번호 입력받기
    - (e) 진행중인 체스방은 삭제 불가능
    - 동일한 비밀번호라면 삭제
